Homebrew Hub

Welcome to Homebrew Hub

Homebrew Hub is a community-led initiative to collect, archive and preserve homebrew software developed for Game Boy (Color).
Each ROM can be easily downloaded or played directly in the browser, through an accurate emulator.
1253 entries are currently in the database.

Games Showcase

Contribute

Anyone can add new entries or improve the existing ones. Whatever it's something you just finished creating with your friends or a lost music cartridge from 20 years ago, check out the database repository to get involved.

Digital Preservation

We are committed to build a digital memory of the Game Boy homebrew development scene. We salvaged some of the first ever unofficial software from the 90s and we keep working to harvest more.

Open and accessible

The database is maintained on a public git repository. Everyone can contribute and audit. Entries metadata follows a well defined JSON schema and the full dataset is released under a permissive license.
